JDeveloper Copyright Plugin

A simple plugin for JDeveloper that will add copyright/license info

This is an attempt to provide similar functionality to Intellij's Copyright Plugin from within JDeveloper.

You can read more in the blog post

Installation

Check for Updates Bundle

  1. Start JDev
  2. Go to Help -> Check for Update -> Add
  3. Enter the following into the add dialog Name: Whatever_You_Want Location: https://raw.github.com/jacobdanner/jdev-copyright-plugin/master/center.xml
  4. Select extension and install
  5. Restart JDeveloper

Local Install

  1. Download the Bundle Jar
  2. Go to Help -> Check for Update -> Install from Local File
  3. Select the file you just downloaded.
  4. Restart JDeveloper

How to Use

The extension is pretty simple once its installed

  1. Go to Tools -> Preferences -> Copyright
  2. Modify the copyright text
  3. Select File(s) in the navigator menu Right Click Select Insert Copyright

That's it :)
